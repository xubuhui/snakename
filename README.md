# SnakeName - Java 游戏项目

一个基于 Java 的贪吃蛇游戏。

## 📋 项目简介

SnakeName 是一个经典的贪吃蛇游戏实现，使用 Java 开发，支持图形化界面。

## 🛠️ 技术栈

- **语言**: Java
- **图形库**: Swing/AWT
- **游戏引擎**: 自定义

## 📦 安装

```bash
git clone https://github.com/xubuhui/snakename.git
cd snakename
javac *.java
```

## 🚀 使用

```bash
java SnakeGame
```

## 🎮 游戏规则

- 使用方向键控制蛇的移动
- 吃到食物增加长度
- 不能撞到墙壁或自己
- 分数越高越好

## 📊 功能

- ✅ 经典贪吃蛇玩法
- ✅ 实时分数显示
- ✅ 难度调整
- ✅ 游戏暂停/继续
